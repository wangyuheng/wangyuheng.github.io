DDL转plantuml格式ER图

Posted on 2018-06-28 Edited on 2020-11-06 In java

项目源码地址: github

背景

通过plantuml绘制数据库ER图，手写字段信息成本太大。需要一套DDL转换plantuml的工具。

方案

  1. 读取同目录下ddl.sql文件
  2. 基于 ; 分割建表语句
  3. 依赖 druid 进行sql解析
  4. 定义plantuml模版
  5. replaceAll替换模版内容

使用

程序会读取同目录的ddl.sql文件，并转换生成er.puml文件。
